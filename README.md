## Launchpad Contract

This repository includes a **Launchpad** smart contract written in Ralph for the Alephium blockchain.

### Features:
- Time-limited token sale  
- Whitelist-based participant access  
- Hard cap on total funds raised  
- Minimum and maximum contribution limits  
- Immediate token transfers upon contribution  
- Emergency pause/unpause functionality  
- Owner-controlled token price updates  
- Owner withdrawal of raised funds after sale  

### Usage
- Contributors send ALPH tokens during the sale period to purchase tokens  
- Only whitelisted addresses can participate  
- Tokens are distributed instantly proportional to contributions  
- Owner can manage sale parameters and withdraw funds
